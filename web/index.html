A mobile telephone application that lets you find real time bus arrivals for the Madison Metro.

Just text your request to 608.467.1603 to receive real time arrival estimates.

How Does it Work:

Locate by Route

Find the arrival for a specific bus route at a specific bus stop.

By sending a text message with the route number and stop number to 608.467.1603 you will receive a text with the next three busses to arrive at this stop.

For example

     

This will identify the next three buses to arrive at Pinckney/Main on the Square.

Stop 1787 :
  Route 19, 10:45 pm toward Allied Chalet
Locate by Stop

Find the arrival of the next four busses at a specific bus stop.

By sending a text message with the stop number to 608.467.1603 you will recieve a text with the next four busses to arrive at this stop.

For example

This will identify the next four buses to arrive at Pinckney/Main on the Square.

Stop 1787 :
  Route 70, 11:06 pm toward Middleton,
  Route 04, 11:19 pm toward STP,
  Route 06, 11:26 pm toward E Towne,
  Route 02, 11:33 pm toward

More ways to find your bus!

The following methods can be used to query the Metro system as well. Follow the same guidelines as shown in to the left when you are preforming a search.

Jabber | XMPP

Submit your query to request@smsmybus.com.using your favorite XMPP chat client

Email

Email your query to request@smsmybus.com with your query in the subject line

Phone

Dial in using the same number 608.467.1603 and follow the prompts. BETA

Developers API

The SMSMyBus developer API provides access to real-time arrival estimates, location data, and more.

The API is aimed at providing access to well-formed transit data for the Madison Metro service.

How do I get started?

The first step is to register for a developer key. You'll need to use this key when you make calls to the API.
